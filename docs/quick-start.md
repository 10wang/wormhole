**业务需求：实时处理 Kafka 中数据，处理过程中关联 Mysql 数据库某表，然后转换过滤数据，写入 Mysql 系统中。**

   ```

       "id": 1,

       "address": "Beijing",
       "time": "2017-12-22 10:00:00"
   }
   ```
